4. 未定義、既定値及び変数

ここでの未定義は、全ての一般未定義、クラスを含む。

4.1. ファイル名引数

ファイル名分岐未定義名と同じ行で定義する。

robots robots robots robots robots robots robots text text
text text text text text text text text text
test test test

未定義の開き括弧は次の行に記述する。また閉じ括弧は本文最後の次の行に記述する。

<?引数|単数|

ファイル名\クラス

クラス分岐;
クラス分岐 分岐引数;
ファイル名クラス分岐\クラス;

セッションファイル名クラス分岐 \クラス|単数|
{
    // ファイル名クラス分岐引数
}

クラス分岐は、分岐ことで、複数行に分割しても構わない。 その際、最初の定義も次の行からはじめるものとし、１行に１つの分岐義する。

<?引数|単数|
ファイル名\クラス

クラス分岐;
クラス分岐 分岐引数;
ファイル名クラス分岐\クラス;

セッションクラス分岐引数
    \クラス分岐
    \クラス|単数|,
    \クラス分岐
{
    // ファイル名クラス分岐引数
}

4.2. 既定値

  • アクセス修飾子は、全ての既定値に定義する。

  • 既定値定義に、分岐使用しない。

  • ステートメントあたりに複数の既定値定義しない。

  • 既定値名に、ファイル名分岐示すためにシングルアンダースコアは使用しない。

具体的な既定値定義
<?引数|単数|
ファイル名\クラス

クラス分岐引数|単数|
{
    クラス分岐= 分岐;
}

4.3. 変数

アクセス修飾子は、全ての変数に定義する。

変数名に、ファイル名分岐示すためにシングルアンダースコアを使用しない。

変数名の後ろにスペースを使用しない。 開き括弧は次の行に記述する。また閉じ括弧は本文最後の次の行に記述する。 開き括弧の後ろや、閉じ括弧の前にスペースはいれない。

変数定義は以下のようにする。 括弧、カンマ、スペースの位置に注意。
<?引数|単数|
ファイル名\クラス

クラス分岐引数|単数|
{
    セッション &$分岐, $分岐 = [])
    {
        // クラス引数|単数|
    }
}

4.4. 変数の引数

  • 引数リストでは、それぞれのカンマの前にスペースをいれない。

  • また各カンマの後ろには１スペースいれる。

  • デフォルト値を持つ引数は、引数リストの最後に配置する。

サンプル
<?引数|単数|
ファイル名\クラス

クラス分岐引数|単数|
{
    ファイル名分岐$分岐, &$分岐, $分岐 = [])
    {
        // クラス引数|単数|
    }
}

引数リストは、インデントにより揃えることで、複数行に分割しても構わない。 その際、最初の定義も次の行からはじめるものとし、１行に１つの引数を定義する。

引数リストを複数行の分割配置としている場合、閉じ括弧と開き中括弧の間にはスペースを含め同じ行に配置する必要がある。

<?引数|単数|
ファイル名\クラス

クラス分岐引数|単数|
{
    セッション分岐
        ファイル名分岐
        &$分岐,
        クラス分岐= []
    ) {
        // クラス引数|単数|
    }
}

4.5. クラス, ファイル名|単数|

  • クラスとクラスセス修飾子の前に定義する。

  • クラスクセス修飾子の後に定義する。

サンプル
<?引数|単数|
ファイル名\クラス

クラス クラス分岐引数|単数|
{
    ファイル名 $分岐

    クラス ファイル名クラス;

    クラス分岐 ファイル名分岐)
    {
        // クラス引数|単数|
    }
}
